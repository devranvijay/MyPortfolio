## üöÄ Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/devranvijay/MyPortfolio.git
    cd MyPortfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
